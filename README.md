# Types:
## Core Types:
    number      2, 3.5, -10
    string      'Hi', "Hi", `Hi`
    boolean     true, false
    object      {age: 27}
    Array       [1, 2, 3]
    Tuple       [1, 2]
    Enum        enum {NEW, OLD}
    Any         *
    Union       string | number
    Function
    unknown     Bit more stricter than any
